# DarkBLstory
黑暗童話心理測驗：測你對黑暗特質的上癮度

玩完歡迎看看我們的書：
當童話遇上BL，加上30個元素標籤，王子與王子的幸福結局被徹底改寫。
這裡有純愛，也有虐戀；有純潔清水，更多的是深夜福利。
儘管風格百變，這10個故事都是不折不扣的暗黑童話。
準備好顛覆你對童話的想像了嗎？來，走進充滿瘋狂與誘惑的黑森林⋯⋯
